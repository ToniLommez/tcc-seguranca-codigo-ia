- Auth:     JWT tokens (HS256 via jwt-cpp) + PBKDF2 password hashing (OpenSSL)
- Database: Firebase Firestore via REST API (Google OAuth2 service account auth)
- Frontend: Single-page application (vanilla HTML/CSS/JS) served by the backend
- Build:    CMake + vcpkg (OpenSSL triplet: x64-mingw-dynamic)
